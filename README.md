# -API-Gares-et-Trains
A robust Spring Boot application designed to interface with the "Gares et Trains" (Stations and Trains) API network. This project provides a centralized testing dashboard to interact with 15 distinct endpoints, facilitating real-time data retrieval and API behavior analysis.
